# OpenFohhn
An open-source controller for Fohhn amplifiers using the documented FDCP protocol. This project is independent and is not affiliated with or endorsed by Fohhn.


# OpenFohhn

Open-source controller for Fohhn amplifiers.

## Features

- USB (FTDI)
- Serial (RS-232)
- UDP (planned)
- Windows
- Linux
- Raspberry Pi
- Home Assistant
- AirPlay integration (planned)

## Supported devices

| Device | Status |
|---------|--------|
| MA-4.100 | ✅ Tested |
| DI Series | 🚧 Planned |
| FI Series | 🚧 Planned |
| Scale | 🚧 Planned |
